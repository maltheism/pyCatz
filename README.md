# pycat - iEEG to BIDS Converter Wizard

pycat is a GUI interface for iEEG to BIDS conversion for LORIS (Longitudinal Online Research and Imaging System).

## Installation

#### System Requirements

 * Linux, macOS, Windows

#### Development Requirements

 * Node.js >= 14.5.4
 * NPM >= 7.5.2
 * Python 3.8
 * PyInstaller 4.2
